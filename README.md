## рҹӣ пёҸ Requirements
- вҳ• Java 17+
- рҹҗҳ PostgreSQL

## вҡҷпёҸ Environment Variables
To run the project, you need to set the following environment variables:

* рҹ‘Ө `DB_USERNAME` - username for connecting to the PostgreSQL database
* рҹ”‘ `DB_PASSWORD` - password for connecting to the database

## рҹҡҖ How to run locally

1.  Clone the repository.
2.  Set the `DB_USERNAME` and `DB_PASSWORD` environment variables in your IDE or terminal.
3.  Run the `UlrShortenerApplication` class.
4.  Swagger documentation will be available at: [http://localhost:8080/swagger-ui/index.html](http://localhost:8080/swagger-ui/index.html)
